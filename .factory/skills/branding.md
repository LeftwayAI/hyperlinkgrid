# Hyperlinkgrid Branding Guidelines

## Core Identity
*   **Name**: Hyperlinkgrid (Capital H, lowercase g) or hyperlinkgrid. Never HyperlinkGrid.
*   **Vibe**: Precision, Blueprint, Technical, "Internet Blueprint". Not "Brutalist" in the heavy sense, but refined and grid-based.

## Color Palette
*   **Primary**: Pure White (`#FFFFFF`)
*   **Text/Borders**: Pure Black (`#000000`)
*   **Accent**: CSS Blue (`#0000FF`) - Used for active states, links, and the logo.

## Typography
*   **Font**: Satoshi (Variable font).
*   **Style**: Clean, precision-oriented, technical.

## Design System Rules
1.  **Precision**: No gradients, no shadows, no rounded corners.
2.  **Grid-Based**: Everything aligns to a rigid grid.
3.  **1px Borders**: All interactive elements (buttons, inputs, cards) have a 1px solid black border.
4.  **Hover States**: Invert colors (Black bg / White text) or Blue bg / White text.
5.  **Inputs**: Standard HTML inputs styled with 1px borders.

## Logo
*   Simple text "Hyperlinkgrid" or a 10x10 pixel grid icon.
