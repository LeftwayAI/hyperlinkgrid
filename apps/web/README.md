# Hyperlinkgrid Web

This folder contains the Next.js app for the Hyperlinkgrid project.

## Getting started

```bash
pnpm install
pnpm dev --filter @hyperlinkgrid/web
```

- `pnpm dev` runs the local development server on http://localhost:3000.
- `pnpm build` generates a production build.
- `pnpm lint` runs the default Next.js lint checks.

> Note: The homepage references `public/logo.svg`. Drop your logo in that file path before shipping a build.
